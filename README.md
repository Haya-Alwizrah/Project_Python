# Game Hub

**Game Hub** is a collection of classic games developed in Python as part of the Tuwaiq Academy AI Bootcamp. The project combines game development concepts with clean object-oriented design and web deployment using Flask.

---

## Live Demo
[Play Game Hub](https://game-hub-skxc.onrender.com)

## Features

- Multiple playable games in one platform
- Single-player and multiplayer modes
- Object-Oriented Programming (OOP) architecture
- Web interface powered by Flask
- Deployment-ready using Render

---

# Available Games:
### Single Player Games:

- Hangman:
A classic word-guessing game where the player tries to discover the hidden word before running out of attempts.

- Wordle:
Guess the secret word within a limited number of tries using letter-position feedback.

### Multiplayer Games:

- Tic Tac Toe:
A two-player strategy game played on a 3×3 board.

- Hexapawn:
A simplified chess-inspired game that focuses on pawn movement and strategic decision-making.
